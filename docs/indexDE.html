TV-Browser Plugin für TVStreamRecord

For english version of this page please use this link.

Dieses plugin erlaubt Ihnen Ihre Lieblingssendung aus dem TV-Browser heraus mit TVStreamRecord auf Ihrem NAS aufzunehmen.

Installation

Bitte benutzen Sie die offizielle Plugin-Verteilung von TV-Browser. Für eine manuelle Installation müssen Sie die letzte Version herunterladen und die .jar-Datei in den Plugins-Ordner von TV-Browser (z.B. C:\Program Files (x86)\TV-Browser\plugins) kopieren.

Konfiguration

Nach der Installation müssen Sie TV-Browser starten, Menü 'Extras' öffnen und 'Plugins verwalten' wählen. Jetzt sollte TsrPlugin in der Liste zu sehen sein. Klicken Sie es an und geben Sie die volle URL von TVStreamRecord inklusive Port ein (z.B. http://0.0.0.0:8030) ohne anschließendem Backslash.

Sie können nun aus dem Kontext-Menü heraus Ihre Lieblingssendung direkt an TVStreamRecord senden und mithilfe des Toolbar-Buttons das Web-Interface aufrufen.

Für das Konfigurieren von TVStreamRecord wenden Sie sich bitte an die entsprechende Anleitung und Hilfeseiten.

Bitte beachten Sie, dass Ihre Sender in TVStreamRecord genau den gleichen Namen haben sollen, wie in TV-Browser angezeigt, um richtig synchronisiert zu werden. Wenn keine Aufnahme erstellt wurde, überprüfen Sie bitte den Log von TVStreamRecord für mehr Information.
